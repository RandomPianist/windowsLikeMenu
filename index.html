1. Introdução

================================
| NÃO COMERCIALIZE ESSE SCRIPT |
================================

WINDOWS LIKE MENU 1.1, por Reynolds Costa

Essa biblioteca tem a finalidade de simular um menu como o do S.O. Windows, incluindo comandos de atalho e seleção com o teclado.

Ela faz uso obrigatório da biblioteca previamente publicada combinarTeclas, sendo requerida a sua versão 2.0.

2. Instalação

Para começar a usar essa biblioteca, copie os arquivos windowsLikeMenu1.1_func.js e windowsLikeMenu1.1_main_mini.js* na pasta js do presente repositório e o arquivo combinarTeclas2.0_main_mini.js* do repositório previamente publicado combinarTeclas e faça conexões com eles no seu projeto usando o atributo src da tag <script>.

A sua instalação deve se parecer com isso:

Ver código

<script type = "text/javascript" language = "JavaScript" src = "js/combinarTeclas2.0_main_mini.js"></script>

<script type = "text/javascript" language = "JavaScript" src = "js/windowsLikeMenu1.1_main_mini.js"></script>

<script type = "text/javascript" language = "JavaScript" src = "js/windowsLikeMenu1.1_func.js"></script>

* Não é necessário usar os arquivos windowsLikeMenu1.1_main.js e combinarTeclas2.0_main.js, mais pesados que sua contrapartes minificadas, disponíveis em seus respectivos repositórios apenas para consulta e contribuição.

3. Utilização

3.1. Objeto principal

O funcionamento dessa biblioteca se baseia em um objeto JSON principal que é instanciado recursivamente de acordo com os filhos na árvore do menu. Para compreender a biblioteca, é necessário compreender antes esse objeto simples. Ele possui as seguintes chaves:

Nome Tipo Obrigatório Explicação
texto string Sim Faz referência ao texto de cada elemento do menu.
letraAlt number Não Índice da posição do caractere no texto que será destacado e poderá ser pressionado no teclado para executar o comando.
Não pode fazer referência a um caractere que já tenha sido declarado ou não seja alfabético.
atalho array Não. Não declare caso hajam filhos. Vetor com os códigos das teclas que, ao pressionadas em conjunto e na ordem, executam a função. Não declare se houverem filhos.
funcao function Sim se não houver filhos, não declare caso hajam. Função que é executada ao se clicar sobre o item no menu ou ao pressionar a combinação de teclas que faz referência ao seu atalho.
destacado boolean Não. Não declare em elementos na raiz. Inclui uma barra vertical acima do item.
desativado boolean Não Desativa o item do menu quando verdadeiro.
filhos array Sim se não houver função, não declare caso hajam. Vetor com uma lista de objetos com a mesma estrutura.

3.3. Função ativar

É disponibilizada a função ativar, que tem a função de ativar ou inativar os itens do menu. Sua sintaxe é a seguinte:

Ver código

WLM.ativar(id, ativo);

Onde id é uma string contendo números em base 0 separados por underlines que representam cada um dos itens do menu e ativo é um boolean que determina se o item deve ser ativado caso true ou inativado caso false.

O botão abaixo a utiliza para alternar a ativação do item 1_1 do menu, isto é, Redes > Outras.

Observe o código deste botão:

Ver código

ativo = this.value == "Desativar";

this.value = !ativo ? "Desativar" : "Ativar";

WLM.ativar("1_1", !ativo);

3.4. Propriedade WLM_letra_auto

A propriedade WLM_letra_auto é um boolean que, caso tenha seu valor alterado para true ANTES da construção do objeto WLM, fará com que a construção do menu ignore todos os índices estabelecidos na propriedade letraAlt de cada elemento e crie para ele um novo índice.

O novo indíce automático é baseado na primeira letra do texto do menu. Isso é feito para todos os elementos. Portanto, caso a próxima primeira letra seja uma repetição, o sistema tentará aplicar o índice à segunda letra e assim por diante. Isso resultará em erro e o menu não será construído caso, ao chegar ao fim do texto do elemento, nenhuma letra tenha sido encontrada disponível para se aplicar o índice. Portanto, recomenda-se utilizar o padrão false (não sendo necessária a declaração) para menus com muitos elementos ou para menus cujos elementos tenham textos relativamente curtos.
