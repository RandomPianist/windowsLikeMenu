Instruções de uso:

================================
| NÃO COMERCIALIZE ESSE SCRIPT |
================================

WINDOWS LIKE MENU, por Reynolds Costa

Esse script tem a finalidade de simular um menu como o do S.O. Windows, incluindo comandos de atalho e de seleção com o teclado.

Ele faz uso do script previamente publicado combinarTeclas.

Para usá-lo, copie os arquivos windowsLikeMenu_func.js e windowsLikeMenu_main_mini.js* do presente repositório e o arquivo combinarTeclas_main_mini.js* do repositório homônimo e faça conexões com eles no seu html nessa ordem usando o atributo src da tag <script>.

Depois disso, abaixo dessa conexão, basta instanciar a classe Menu segundo a sintaxe demonstrada em exemplo.js.

* Não é necessário usar os arquivos windowsLikeMenu_main.js e combinarTeclas_main.js, mais pesados que suas partes minificadas, disponíveis em seus respectivos repositórios apenas para consulta e para contribuição.
